Welcome to Earthworm

Your guide to easy Python scripting

Introduction

Earthworm is a Python module designed to make scripting easier for beginners. With a wide range of simple commands, Earthworm helps you perform common tasks without needing to memorize complex syntax.

Getting Started

To start using Earthworm, you need to import it into your Python script. Hereâ€™s how you can do it:

Once imported, you can start using the various commands provided by Earthworm.

Examples

Here are some examples of how to use Earthworm in your Python scripts:

Example 1: Simple Arithmetic

import earthworm as ew

result = ew.add(5, 3)
ew.output(f'The sum of 5 and 3 is {result}')

Example 2: User Interaction

import earthworm as ew

name = ew.ask("What is your name? ")
ew.output(f'Hello, {name}!')

Example 3: File Operations

import earthworm as ew

ew.write_file("example.txt", "This is an example.")
content = ew.read_file("example.txt")
ew.output(content)
